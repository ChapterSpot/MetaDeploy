# UserInfo

** Shows information about the Salesforce Org during an installation Job.**

## Overview

The **UserInfo** component is an
[SLDS Card](https://react.lightningdesignsystem.com/components/cards/) used to
display information about the Salesforce Org used to install a **Job**.

## States

If the User connects with a Scratch Org, the card shows the expiration date of
the Org and displays a message regarding what happens when the Org expires.

<Story
  id="jobs-userinfo-example--user-info-component&args=scratchOrg:With+Scratch+Org"
  height="250px"
/>

If the User is connected with a different type of Org, the heading displays a
User profile icon and reads "Salesforce Org Information."

<Story
  id="jobs-userinfo-example--user-info-component&args=scratchOrg:Without+Scratch+Org"
  height="250px"
/>

If provided, information for User name, Org name, and Org type will be shown.
